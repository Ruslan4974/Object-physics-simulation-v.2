Продемонстрирована симуляция роевого движения птиц (стаи).

Каждая птица летит под своим углом и с одной скоростью.
При подлёте к друг другу они максимально подстраивают
свой угол с другими птицами. Меняется соответсвенно направление.
Происходит это в определенной зоне с радиусом R.
Количество птиц n.

Параметры:
    v0 - скорость
    eta - случайный угол отклонения (в радианах)
    L - размер области
    R - радиус взаимодействия
    dt - фаза
    Nt - количество фаз
    N - количество частиц
    mean_theta = (theta)i ∈ R - средний угол направления всех соседей
птицы i внутри R.

Обновление положения птиц на каждой фазе: xi = xi + v0(cos(theta))*dt
					  xi = xi + v0(sin(theta))*dt

Обнволение движения с постоянной скоростью: vi = v0(cos(theta))
					    vi = v0(sin(theta)).

Динамичность всей модели: theta = (theta)i ∈ R + eta i, где
eta получено в результате равномерного распределения [-η/2,η/2].
